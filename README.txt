# ECGM-Iris_Inclusiva
Iris Inclusiva


Passos para correr o projeto:
 
-Criar base de dados: gestaoassociados

- open in the terminal the project location

backend (Laravel):
- make sure you have composer installed
- composer install
- to run the project: php artisan serve

- open frontend folder

Front (React):
 - npm install
 - npm run start
